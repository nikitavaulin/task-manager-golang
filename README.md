# Task-manager 
**REST API** приложение, реализующее функциональность планировщика задач. Используется трехзвенная архитектура **transport-service-repository**.

Является дипломной работой курса Яндекс Практикума: *"Go-разработчик с нуля"*.

## Функциональность
1. Добавление / Изменение / Удаление задач;
2. Настройка правил повторения задач;
3. Выполнение задач;
4. Поиск задач по дате / заголовку / описанию;
5. Авторизация пользователей.

## Выполнение заданий повышенной трудности
1. Чтение переменных окружения (порт, БД, пароли, ключ для jwt-токена);
2. Правила повторения для месяца и недель;
3. Поиск задач по дате / заголовку / описанию;
4. Авторизация пользователей (с использованием jwt, bcrypt);
5. Создание docker-образа.

## Запуск
1. Задайте переменные окружения согласно [.env.example](.env.example).
```
HOST_PORT=
TODO_PORT=
TODO_DBFILE=
HTTP_SHUTDOWN_DURATION=
TODO_PASSWORD=
JWT_SECRET=
```

2. Запустите приложение.

### Локальный запуск
```bash
go run cmd/main.go
```

### Запуск через docker
```bash
docker compose up -d
```

## Тестирование
1. Запустите приложение.

2. Задайте параметры в файле [settings.go](tests/settings.go), следуя инструкциям ниже.
```go
var Port = 8080 // Ваш HOST_PORT или TODO_HOST (если запускали локально)
var DBFile = `../scheduler.db` // Ваш TODO_DBFILE
var FullNextDate = true 
var Search = true
var Token = `` // Полученный токен в cookie после авторизации 
```

2. Запустите тесты.
```bash
go test ./tests
```
